Power Tools

Power Tools in Potentia are super-charged tools with no durability. However, they require power to operate. To fill a power tool with energy, you must place it in a recharger. Each of the tools has a maximum capacity of 5000 Tool Energy. Each and every time you break a block/hit an entity the power tool will lose one tool energy.
