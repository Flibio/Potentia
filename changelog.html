Changelog

Version 1.0.0:

  • Breaking blocks bug fixes
  • Generator bug fixes
  • Balance generator power rate to: uses 1 coal every 2 seconds, producing 3 Machine Energy/tick. This makes 120 Machine Energy per coal.

Version 1.0.0 - Pre 2:

  • Sign Linker bug fixes
  • Ability to clear Sign Linker by right-clicking air with it in hand
  • Generator will stop you from placing block on it unless you are sneaking
  • Generator message format changes
  • Generator now has a maximum fuel capacity of 1,000,000
  • Generator now accepts coal blocks as fuel
  • Generator now draws coal, charcoal, and coal blocks from adjacent chests every 5 seconds
  • Recharger message format changes + a new metric
