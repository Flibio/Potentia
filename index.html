Introduction

Potentia is a plugin that adds power, machines, and more to any Minecraft server running Bukkit/Spigot. Potentia requires no mods to run, yet adds many mod-like features!

Installing Potentia

After you have downloaded the latest version of Potentia, you are ready to begin installation. There are no configuration files to edit, so installation is a breeze!

  1. Stop your server if it is currently running
  2. Drag and drop the Potentia JAR file into your plugins folder
  3. Start your server again
  4. Installation is complete!

Getting Started

To get started with Potentia you are going to want to craft a generator. This will generate and store power that will be used by Potentia machines. After you craft a generator, you will need to place it somewhere in the world. Potentia will send you a message that says the generator was placed. The generate generates power using coal that you place in it. to view the coal and power supplies that the generator contains simply right-click it. To start generating power, you will need to insert coal into the generator by right-clicking on it with coal in your hand. Then supply redstone power to the generator and it will begin to generate power!
